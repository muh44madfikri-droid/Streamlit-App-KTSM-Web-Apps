# 🤝🏻 Karang Taruna Suko Manunggal - Arisan System

A web-based application built using **Streamlit** to manage arisan activities and support community administration for Karang Taruna Suko Manunggal.

## 🌐 Overview

This application is designed for the youth organization (Karang Taruna) of RT 13, RT 14, and RT 20 Dukuh Bandungan. It serves as a platform for:

- Managing arisan data
- Recording member contributions
- Providing an organizational profile page
- Supporting transparency and collaboration among members

## ✨ Features

- 🏠 Landing / Profile page for organization introduction  
- 👥 Member management  
- 💰 Arisan tracking system  
- 📊 Simple data visualization  
- 🔐 Multi-page navigation using Streamlit  
- 📱 Responsive and accessible via web browser  

## 🎯 Purpose

To help Karang Taruna Suko Manunggal improve:
- Organizational management  
- Financial transparency  
- Member engagement  
- Digital transformation of community activities
